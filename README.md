# MAYOT Partner Investment Landing

Односторінковий сайт + друга сторінка для deep research пояснення партнеру інвест-проєкту з інтерактивними калькуляторами:

- калькулятор залежності ціни від площі, локації та попиту;
- калькулятор витрат, прибутку і ROI;
- окрема сторінка `research.html` зі сценарним порівнянням (консервативний/базовий/оптимістичний).

## Запуск локально

```bash
python3 -m http.server 4173
```

Потім відкрийте:
- `http://localhost:4173/` — головна сторінка;
- `http://localhost:4173/research.html` — друга сторінка deep research.
