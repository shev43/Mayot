# MAYOT Presentation Site

Сайт-презентація для партнерського пітчу MAYOT з фінансовим симулятором.

## Що всередині

- структура презентації по секціях (проблема, рішення, модель, план, CTA);
- sticky-навігація для швидкого переходу між «слайдами»;
- інтерактивний блок фінансових розрахунків (виручка, витрати, прибуток, ROI).

## Локальний запуск

```bash
python3 -m http.server 4173
```

Відкрийте: `http://localhost:4173`
